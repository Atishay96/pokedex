pokedex
==

Description
==
Pokedex is database for pokemons.

Installation
==

1. `git clone https://github.com/Atishay96/pokedex`

2. `npm install`

3. `npm start`


if error occurs while install it via npm follow the steps below.

1. `git clone https://github.com/Atishay96/pokedex`

2. `npm install yarn -g`

3. `yarn install`

4. `yarn start`
